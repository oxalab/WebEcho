<h1 style="text-align: center;">WebEcho</h1>


<h4 style="text-align: center;">Copy websites to your computer! WebEcho is a utility that allows you to download a website from the Internet to a local directory. Get html, css, js, images, and other files from the server to your computer. WebEcho arranges the original site's relative link-structure. Simply open a page of the "mirrored" website in your browser, and you can browse the site from link to link as if you were viewing it online.</h4>
